PyOBD Package Build and Installation Instructions

Obtaining PyOBD Source Code

The latest version of PyOBD can be obtained from github at https://github.com/beardedone55/pyobd/.

PyOBD Package Build

PyOBD can be packaged into a Python Wheel or a Debian Package to simplify software distribution and installation.

Building PyOBD Python Wheel

A Python Wheel is an installation package that can be installed on any system with Python and pip.

Prerequisites

Python 3.x and setuptools are required to build Python Wheel for PyOBD.

Python Wheel Build Instructions

Enter the following command to generate a Python Wheel for PyOBD:

python3 setup.py bdist_wheel

Or if you have GNU make installed:

make wheel

The resulting .whl file can be found in the dist/ subdirectory.

Building PyOBD Debian Package

A Debian package is an installation package that can be installed on Debian Linux or any derivative, such as Ubuntu.

Prerequisites

Python 3.x, setuptools, GNU make, and debhelper are required to build a Debian package for PyOBD.

Debian Package Build Instructions

Enter the following command to generate a Debian package for PyOBD:

make deb

The resulting .deb file can be found one level up from the pyobd directory (../).

Installing PyOBD

PyOBD can be installed from a Python Wheel, Debian package, or directly from source. Generation of a Python Wheel or Debian package from source is discussed above.

Installing PyOBD from Python Wheel

The Python Wheel is a platform independent package type that can be installed on any system that has Python and PIP.

Prerequisites

Python 3.x and pip3 are required to install PyOBD from a Python Wheel.

PyOBD Python Wheel Installation Instructions

Enter the following command to install the Python Wheel:

pip3 install pyobd_beardedone55-0.9.4-py3-none-any.whl

This will install PyOBD along with any other required packages, such as PyQt5 and PySerial.

Installing PyOBD from Debian Package

The Debian package is a package type that can be installed on any system that is running a Debian Linux or any derivative, such as Ubuntu.

Prerequisites

Python 3.x, PyQt5, and pyserial, are required to install PyOBD from a Debian package.

PyOBD Debian Package Installation Instructions

Enter the following command to install the Debian package:

sudo dpkg -i python3-pyobd-beardedone55_0.9.4_all.deb

This will not install any packages that are required by PyOBD, such as PyQt5 and pyserial. The installation will fail if the required packages are not already installed.

Installing PyOBD from Source

PyOBD may also be installed directly from the source.

Prerequisites

Python 3.x and setuptools are required to install PyOBD from source.

PyOBD Source Installation Instructions

To install for all users, enter the following command from the source directory:

sudo python3 setup.py install

To install for the current user only, enter the following command from the source directory:

python3 setup.py install --user

This will install PyOBD along with any other required packages, such as PyQt5 and PySerial.
